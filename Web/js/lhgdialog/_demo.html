lhgdialog - samples

1. 不带遮罩层的普通窗口


调用代码：lhgdialog.opendlg( '不带遮罩层的普通窗口', '/lhgdialog/_content/content1.html', 400, 300 );

2. 带遮罩层的普通窗口


调用代码：lhgdialog.opendlg( '带遮罩层的普通窗口', '/lhgdialog/_content/content1.html', 400, 300, true );

3. 内容页为外部连接的窗口


调用代码：lhgdialog.opendlg( '内容页为外部连接的窗口', 'http://www.google.com', 640, 480, '', true );

4. 内容页为html代码的窗口


调用代码：lhgdialog.opendlg( '内容页为html代码的窗口', '<div style="color:red;font-size:14px;">lhgdialog弹出窗口</div>', 400, 300, '', 'hcode' );

5. 带遮罩层带子窗口的窗口


调用代码：lhgdialog.opendlg( '带遮罩层带子窗口的窗口', '/lhgdialog/_content/content2.html', 400, 300, true );
子窗口的调用代码（写在content2.html里）：D.lhgdialog.opendlg( '我是子窗口', '', 400, 300, true );

6. 在调用页面按钮关闭弹出窗口


调用代码：lhgdialog.opendlg( '被调用页面关闭的窗口', '/lhgdialog/_content/content1.html', 400, 300, false, false, '', '', 'win1' );
关闭窗口按钮代码：lhgdialog.closdlg('win1');

7. 在指定位置弹出窗口


调用代码：lhgdialog.opendlg( '在指定位置弹出窗口', '/lhgdialog/_content/content1.html', 400, 300, false, false, 300, 100 );
注：可以单独指定距左边位置或距顶部位置

8. 返回值到调用页面


调用代码：lhgdialog.opendlg( '返回值到调用页面', '/lhgdialog/_content/content3.html', 400, 300 );
注：返回值的代码都在conent3.html里，你可打开这个文件查看

9. 多窗口之间的传值


调用代码：窗口1：lhgdialog.opendlg( '窗口1', '/lhgdialog/_content/content4.html', 400, 300, false, false, '', '', 'dlg1');
窗口2：lhgdialog.opendlg( '窗口2', '/lhgdialog/_content/content5.html', 400, 300, false, false, '', '', 'dlg2' );
注：传值操作的代码都在content4.html和content5.html里，可打开查看。

10. 创建其它按钮


调用代码：lhgdialog.opendlg( '创建其它按钮', '/lhgdialog/_content/content6.html', 400, 300 );
注：创建其它按钮的方法都在content6.html里，可打开查看。
